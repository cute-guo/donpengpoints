## 🚀 云端部署方案（发给同事使用）

### 方案一：腾讯云 COS 静态托管（推荐）

**步骤：**
1. 登录 [腾讯云控制台](https://console.cloud.tencent.com/)
2. 进入 **对象存储 COS** → 创建存储桶（设置为"公有读私有写"）
3. 开启 **静态网站托管**功能
4. 将 `dongpeng-crm/` 文件夹下所有文件上传到存储桶根目录
5. 访问 COS 提供的域名（如：`https://xx.cos.ap-guangzhou.myqcloud.com/index.html`）

**优点：** 零成本（5G免费）、快、稳定、国内访问速度快

---

### 方案二：Cloudflare Pages（免费）

**步骤：**
1. 注册 [Cloudflare Pages](https://pages.cloudflare.com/)
2. 创建项目 → 选择"直接上传"
3. 将 `dongpeng-crm/` 文件夹上传
4. 自动获得 `.pages.dev` 域名

**优点：** 完全免费、全球CDN、自动HTTPS

---

### 方案三：微信云开发静态托管

**步骤：**
1. 在微信开发者工具中，进入**云开发 → 静态托管**
2. 上传 `dongpeng-crm/` 目录
3. 获取访问链接即可分享

## 📊 数据更新方式

### 方式一：上传 Excel 文件（网页端操作）

1. 登录系统 → 左侧点击「**数据更新**」
2. 点击「选择文件上传」或拖拽 Excel 到上传区域
3. 系统自动解析并更新所有图表

**Excel格式要求：** 第一行为标题行，包含以下列名（中文，顺序不限）：
- `区域` / `大区` — 区域名称
- `积分预算` / `年度预算` — 积分总预算（元）
- `联合营销` / `全国大促` — 联合营销类目（元）
- `对赌` — 对赌类目（元）
- `店中店装修补贴` / `装修` — 装修补贴（元）
- `进场费支持` / `进场费` — 进场费（元）
- `年度销售额` / `销售额` — 年度销售额（万元）
- `H1目标` / `上半年目标` — H1目标（万元）
- `业绩占比` — 业绩占比（%）

### 方式二：发给 Candy（数据处理仓模式）

直接将最新表格截图或文字发给 Candy，Candy 帮你解析更新数据。

## ⚠️ 注意事项

1. **数据存储**：数据保存在浏览器本地（localStorage），同事访问看到的是各自浏览器的数据副本。
2. **多人共享数据**：若需真正多人实时共享，需要接入后端数据库服务（可联系 Candy 升级）。
3. **数据备份**：重要数据建议定期通过 Candy 导出备份。
